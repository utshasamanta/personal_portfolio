Event Mate

Event Mate is an android app that makes signing up for community events fair and stress-free. Instead of racing to register, users join a waiting list and are randomly selected when spots open. Organizers can create events, upload posters, manage participants, and notify winners - all from within the app. With QR code access, personalized profiles, and optional location-based entry, Event Mate ensures accessibility and equal opportunity for everyone.
